**Grade Decomposition of the Lattice QCD Pion Correlator in the Cl(6) Framework**

*Todd M. Firestone*

## Overview

This repository contains the paper, production code, and verification scripts for a study of the lattice QCD pion correlator recast inside the Clifford algebra Cl(6). The central object is a set of **grade fractions** — projections of the pion correlator onto the four irreducible representation sectors of Cl(6) under its su(3) ⊕ u(1) decomposition — and what those fractions can and cannot reveal about the underlying gauge dynamics.

The Cl(6) framework assigns the eight-component Wilson–Dirac spinor (four Dirac × two color-triplet/singlet sectors) to a single minimal left ideal of Cl(6), with the su(3) color group embedded as the stabilizer subgroup of a preferred spinor direction inside the 15-dimensional bivector algebra. The pion correlator then decomposes into four algebraically distinct channels with exact per-configuration completeness.

---

## Mathematical Framework

The algebra Cl(6) is generated by six orthonormal basis vectors {e₁, …, e₆} satisfying {eᵢ, eⱼ} = 2δᵢⱼ in Euclidean signature. Its 64-dimensional graded structure supports a natural (Γ₇, H) double grading:

| Label | su(3) rep | Γ₇ | H | dim |
|---|---|---|---|---|
| **3** | triplet | +1 | +1/2 | 3 |
| **3̄** | anti-triplet | −1 | −1/2 | 3 |
| **1ₐ** | singlet | +1 | −3/2 | 1 |
| **1_b** | singlet | −1 | +3/2 | 1 |

where Γ₇ = iI₆ is the Spin(6) chirality (I₆ = e₁e₂e₃e₄e₅e₆, anti-Hermitian, I₆² = −I) and H = B₁₂ + B₃₄ + B₅₆ is the Cartan hypercharge of spin(6). The grade fractions are

$$f_r(t) = \frac{C_\pi^{(r)}(t)}{C_\pi(t)}, \qquad \sum_r f_r(t) = 1 \text{ (per configuration)}.$$

### Key Results

**Kinematic Limit Theorem (Theorem 6.1).** In the free Wilson–Dirac theory, $f_r = \dim(r)/8$ exactly: **3/8, 3/8, 1/8, 1/8**. The proof proceeds through the I₄⊗U(2) symmetry of the free propagator, Schur's lemma, free-field color triviality, and the spatial-cubic invariance of the spatially-summed correlator — the residual time-direction term being removed by its vanishing grade-basis diagonal.

**Proposition 6.2 (Spin(6) chirality is dynamically frozen).** On every gauge configuration and at every timeslice,

$$f_{(\Gamma_7=+1)}(t) = f_{(\Gamma_7=-1)}(t) = \tfrac{1}{2}$$

exactly (residual ≤ 3.9 × 10⁻¹⁶ across all configurations and timeslices, both β values). The proof: e₅, e₆ never enter D_W; the gauge links act on color only; so the third Clifford qubit is inert, the time-direction term in the grade-diagonal is zero, and Tr[Γ₇ Q̃] = 0 identically per configuration — free or interacting, quenched or unquenched. Consequently A_color = −A_singlet exactly per configuration, and the only dynamically resolvable observable is the within-Γ₇ H-splitting **f_triplet = f₃ + f₃̄**.

**Bivector su(3) closure (Appendix B).** The bivector su(3) octet — derived as the stabilizer of the H = −3/2 eigenstate |ξ₀⟩ within the 15-dimensional bivector space — is verified to close as a genuine nonabelian Lie subalgebra of spin(6) directly in the 8×8 representation (closure residual 6.2 × 10⁻¹⁶; centralizer = span{H} = span{(3/2)Q_{B−L}}).

## Usage

### Running the production computation

```bash
python cl6_block5_step4b_production.py
```

This performs the full computation on two quenched SU(3) ensembles (β = 4.5 and β = 6.0, N = 6⁴ lattice, m₀ = 0.05, N_cfg = 20 configurations each) and writes `cl6_block5_results.pkl`. The run uses exact point-to-all propagators (24 spin–color source components, single CG solve per configuration). Mean CG iteration counts: 18.95 (β = 4.5), 21.85 (β = 6.0); zero solver failures.

**Note:** The pickle file is included in this repository as the ground-truth reference. Re-running the production script should reproduce all table values and residuals.

### Verifying the bivector su(3) structure

```bash
python cl6_su3_centralizer_check.py
```

Constructs the 15 bivectors Bᵢⱼ = (i/4)[eᵢ, eⱼ] in the 8×8 representation, identifies the su(3) octet as the stabilizer of |ξ₀⟩, and verifies Lie-algebra closure, nonabelianness, and the centralizer identification. All checks should return PASS with the residuals reported in Appendix B of the paper.

## Notes on Design Choices

**Imported vs. emergent color.** The SU(3) gauge group is implemented as the standard 3×3 unitary matrices acting on the ℂ³ color factor of the tensor product ℂ⁸ ⊗ ℂ³ (Stance 2 / factorized architecture). The bivector su(3) inside Cl(6) is a separate representation-theoretic object; both the abstract isomorphism and the explicit bivector closure are verified, but the gauge links are imported, not derived.

**Source types by measurement.** Three distinct source setups are used: (1) exact point-to-all solves for the Experiment A grade fractions; (2) Z₂ stochastic sources for κ_c / PCAC determination; (3) a single Z₂ source for free-field and completeness diagnostics (valid there because those identities are source-independent).

**Bibliography numbering.** Reference numbers [12]–[19] are intentionally reserved to preserve citation-number alignment with Paper 1.
